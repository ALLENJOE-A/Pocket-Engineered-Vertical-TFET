# Pocket-Engineered Vertical Tunnel Field-Effect Transistor (VTFET)

## Overview
This project focuses on the design, simulation, and analysis of a Pocket-Engineered Vertical Tunnel Field-Effect Transistor (VTFET) using Synopsys Sentaurus TCAD. The objective is to improve device performance by optimizing the pocket-engineered structure to achieve higher ON-state current while maintaining low OFF-state current and excellent switching characteristics.

## Objectives

- Design a Pocket-Engineered Vertical TFET structure.
- Simulate the device using Synopsys Sentaurus TCAD.
- Analyze electrical characteristics under different bias conditions.
- Evaluate key performance metrics.
- Study the impact of structural and material parameters on device performance.


## Device Parameters Analyzed

- ON-State Current (ION)
- OFF-State Current (IOFF)
- Threshold Voltage (VTH)
- Subthreshold Swing (SS)
- Transfer Characteristics (ID-VG)
- Output Characteristics (ID-VD)


## Tools Used

- Synopsys Sentaurus TCAD
- Sentaurus Structure Editor (SDE)
- Sentaurus Device (SDevice)
- Sentaurus Visual
